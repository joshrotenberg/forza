## Review: issue #530

### Summary

The implementation adds Claude CLI version checking at startup by leveraging
`claude_wrapper::CliVersion`. Two functions are modified in `crates/forza/src/deps.rs`:

1. `check_agent()` -- now captures stdout, parses the version for `"claude"`, and
   emits an `eprintln!` warning if below `MIN_CLAUDE_VERSION` (2.0.0). Non-claude
   agents and parse failures are silently ignored (presence check only). This is a
   soft warning, not a hard error, matching the plan.

2. `doctor_agent()` -- now parses the version for `"claude"` and surfaces it in the
   `CheckResult` detail string with pass/fail based on the minimum version.
